# Booke-博书UI

## 项目简介

Booke-博书UI

## 技术栈

- **Vue 3**
- **TypeScript**
- **Arco Design Pro**

## 项目创建

要创建此项目，请按照以下步骤操作：

1. 使用以下命令初始化 Vue 项目：

   ```bash
   npm init vue@latest

2. 安装所需依赖：

   ```bash
   npm install --save-dev @arco-design/web-vue
   npm i axios
   npm i mockjs
   npm i @types/mockjs
   npm install sass --save-dev

3. 拉取代码 
4. 项目启动：

   ```bash
   npm run dev
